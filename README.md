# 🪨📄✂️ Rock Paper Scissors Telegram Bot

Простой Telegram-бот для игры в «Камень, ножницы, бумага», написанный на Go.
Бот работает только в личных сообщениях, для каждого пользователя ведётся индивидуальный счёт.

## 🚀 Возможности

- Игра с ботом один на один.

- Три кнопки в интерфейсе: 🪨 Камень, 📄 Бумага, ✂️ Ножницы.

- Случайный выбор бота и мгновенное определение результата.

- Персональная статистика побед, поражений и ничьих для каждого игрока.

- Две команды:

    - `/start` – приветствие и инструкция, запуск игры.

    - `/reset` – сброс счёта текущего пользователя.

## 🛠 Используемые технологии

- [Go 1.25](https://go.dev/)

- [Библиотека для Telegram Bot API v5.5.1](https://github.com/go-telegram-bot-api/telegram-bot-api)

## ⚙️ Установка и запуск

1. Клонируйте репозиторий

```bash
git clone https://github.com/CondoR-R/rock_paper_scissors_tg_bot.git
cd rock-paper-scissors-tg-bot
```

2. Получите токен бота

Напишите [@BotFather](https://t.me/BotFather) в Telegram, создайте нового бота и скопируйте выданный токен.

3. Создайте файл `api_key.go`

В корне проекта создайте файл `api_key.go` (он уже перечислен в `.gitignore`, чтобы токен случайно не попал в репозиторий).  
Содержимое файла:

```go
package main

const ApiKey = "ВАШ_ТОКЕН"
```

4. Установите зависимости и соберите проект

```bash
go mod tidy
go build -o bot
```

5. Запустите бота

```bash
./bot
```

## 🎮 Как играть

- Откройте чат с ботом и отправьте команду `/start`.

- Появится приветственное сообщение и клавиатура с тремя кнопками:  
    - 🪨 Камень
    - 📄 Бумага"
    - ✂️ Ножницы

- Нажмите на одну из кнопок – бот случайным образом выберет свой вариант и сразу сообщит результат (победа, поражение или ничья).

- Счёт автоматически обновляется.

- Чтобы обнулить свой счёт, отправьте команду `/reset`.

## 🔒 Безопасность

Токен бота хранится в отдельном файле `api_key.go`, который исключён из репозитория через `.gitignore`.
**Никогда** не публикуйте этот файл и не делитесь своим токеном.
